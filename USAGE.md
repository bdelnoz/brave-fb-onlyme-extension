# Guide d'utilisation

## Objectif
Cette extension aide à modifier en masse la confidentialité des publications Facebook vers **"Moi uniquement"**.

## Installation rapide
1. Ouvrir `chrome://extensions` (ou `brave://extensions`).
2. Activer le **mode développeur**.
3. Cliquer sur **Charger l'extension non empaquetée**.
4. Sélectionner le dossier du projet.

## Utilisation
1. Ouvrir Facebook et se connecter.
2. Cliquer sur l'icône de l'extension.
3. Lancer l'action depuis la popup.
4. Vérifier les changements de confidentialité appliqués.

## Dépannage
- Si rien ne se passe, recharger la page Facebook.
- Vérifier les permissions dans `manifest.json`.
- Utiliser `diagnose.js` pour diagnostiquer les sélecteurs/flows cassés.
