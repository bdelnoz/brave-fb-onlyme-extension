# Installation Guide

## Requirements

- Brave or Chrome with extension developer mode enabled.
- Access to Facebook in the browser tab where extraction will run.

## Install Steps

1. Open `brave://extensions` (or `chrome://extensions`).
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select the `fb-post-extractor-50` folder.
5. Pin the extension if desired for easier access.

## Update Steps

1. Pull the latest repository changes.
2. Go back to the extensions page.
3. Click **Reload** on the extractor extension card.
4. Re-open Facebook tab before starting a new extraction session.
